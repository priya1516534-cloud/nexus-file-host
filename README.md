# 🚀 Cyber-Nexus Telegram File Hosting Website

100% Real & Working cloud storage script with Admin Panel dashboard.

## 🛠️ Project Setup Guide

1. **GitHub Upload:** Create a repository and add all 10 files.
2. **Hosting Platform (Render/Heroku):** Connect your GitHub Repo.
3. **Secret Environment Variables:**
   - `BOT_TOKEN` = Get from Telegram @BotFather
   - `CHANNEL_ID` = Your Telegram Channel ID (e.g., -100xxxxxxxxx)

## 🔐 Admin Panel Details
- **Route:** `https://your-site-link.com/admin`
- **Default Password:** Change it inside `config.py` file.
